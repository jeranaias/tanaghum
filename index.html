How It Works

Full transparency into the technology behind every lesson

  1. 1

    Choose Content

    Paste a YouTube URL, search for authentic Arabic content, or upload your own audio file. Supports MSA and major dialects. You select the content — the tool never downloads or stores videos.

  2. 2

    Transcription

    If the video has existing captions, those are used directly. Otherwise, the audio is captured through your browser's built-in screen sharing API and transcribed locally using OpenAI's Whisper model. Audio is processed in-memory and never uploaded to external servers.

  3. 3

    AI Generates Exercises

    Google Gemini generates pedagogically-structured comprehension questions across pre-listening, while-listening, and post-listening phases. The transcript text is sent to the API — never audio data. All API calls are made through a Cloudflare Worker proxy.

  4. 4

    Teach & Share

    Export self-contained HTML lessons that work offline, share directly with students via link, or publish to the community gallery. Lessons reference the original YouTube video — they do not contain copies of the source material.

Technical Transparency

Open source, auditable, and designed with privacy in mind

No Data Collection

No user accounts, no analytics tracking, no cookies. Audio processing happens entirely in your browser. The only external calls are for YouTube metadata and LLM text generation.

Fully Open Source

Every line of code is publicly auditable on GitHub. The entire application — frontend, worker proxy, and extraction service — is transparent and open for review.

Reference-Only Lessons

Generated lessons embed YouTube's official player to reference original content. No audio or video files are stored, copied, or redistributed. Content remains on YouTube's servers.

Legal & Fair Use

Designed for educational use within established legal frameworks

Educational Purpose

Tanaghum is an educational tool for Arabic listening comprehension, intended for language instructors and students at academic institutions including DLIFLC and similar programs. It creates pedagogical exercises — not copies of source material.

Fair Use, Content Access & Limitations

Fair Use (17 U.S.C. § 107): Nonprofit educational use. Transformative — creates original exercises, not reproductions. Uses factual, publicly broadcast content. Lessons reference content via YouTube embeds. No files copied or redistributed. May increase viewership of original content.

Content Access: Primarily uses existing captions (text only). Fallback uses browser getDisplayMedia() API (same as Google Meet/Zoom) for in-memory audio transcription. Generated lessons use YouTube's official embed player.

This tool does NOT: download/store copyrighted media, redistribute YouTube content, circumvent DRM, collect personal data, or bypass YouTube access controls.

Open Source: Complete source code is publicly auditable on GitHub.
